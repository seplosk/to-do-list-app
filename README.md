## Introduction

The Todo List Application is a web-based task management tool built with Angular and Tailwind CSS. It allows users to register, log in, add tasks, mark them as completed, and delete them.

## Features

- User registration and login
- Add, edit, and delete tasks
- Mark tasks as completed
- View task details
- Persistent storage using local storage

## Technologies Used

- Angular
- Tailwind CSS
- TypeScript
- HTML
- CSS

## Installation

To set up the project locally, follow these steps:

1. **Clone the repository**:
    ```bash
    git clone <repository-url>
    cd to-do-list-app
    ```

2. **Install dependencies**:
    ```bash
    npm install
    ```
    
## Running the Project

To run the project locally:

1. **Start the development server**:
    ```bash
    ng serve
    ```

2. **Open the application**:
    Open your browser and navigate to `http://localhost:4200`.

## Usage

1. **Register a new user**:
    - Navigate to the registration page and fill in the required details (username, email, password).
    - Click the "Register" button.

2. **Log in**:
    - Navigate to the login page and enter your credentials (email, password).
    - Click the "Login" button.

3. **Add a task**:
    - Enter the task title and optional description in the input fields.
    - Click the "Add Task" button.

4. **Mark a task as completed**:
    - Click the checkbox next to the task title.

5. **View task details**:
    - Click the "More Info" button to expand the task details.

6. **Delete a task**:
    - Click the "Delete" button next to the task.

7. **Log out**:
    - Click the "Logout" button.
